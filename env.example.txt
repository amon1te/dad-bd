Password gate configuration (Vite):

Option A (simple): plaintext password (note: visible in built JS)
  VITE_SITE_PASSWORD=change-me

Option B (preferred): SHA-256 base64 of the password
  VITE_SITE_PASSWORD_HASH=...

Generate the hash:
  node -e "crypto=require('node:crypto'); console.log(crypto.createHash('sha256').update('change-me').digest('base64'))"
